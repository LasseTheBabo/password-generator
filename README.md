# Password Generator

Just a small GUI password generator written in C++ and Qt.

## Build Instructions
```
git clone https://github.com/LasseTheBabo/password-generator.git
cd password-generator
cmake .
make
./PasswordGenerator
```

## Dependencies
- Qt 5 or 6
- CMake 3.10 or higher
- g++ or any C++ 17 compatible compiler
